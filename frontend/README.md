# NewFrontend

Visual-only frontend components built with React and Tailwind CSS.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/
│   ├── ui/          # Pure UI components
│   ├── dashboard/   # Dashboard components
│   ├── layout/      # Layout components
│   └── common/      # Common components
├── pages/           # Page components
├── contexts/        # Theme context
└── App.jsx          # Main app
```

## Tech Stack

- React 18
- Tailwind CSS
- Vite
- React Router
